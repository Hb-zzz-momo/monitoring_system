# 快速开始指南

## 运行应用

### 1. 安装 Flutter
如果还没有安装 Flutter，请访问：https://flutter.dev/docs/get-started/install

### 2. 检查环境
```bash
flutter doctor
```

### 3. 获取依赖
```bash
cd monitoring_system
flutter pub get
```

### 4. 运行应用
```bash
# 在模拟器或真机上运行
flutter run

# 或者指定平台
flutter run -d chrome        # Web浏览器
flutter run -d android       # Android设备
flutter run -d ios           # iOS设备 (仅Mac)
```

## 使用演示

### 登录
1. 应用启动后会显示启动页（2秒）
2. 自动跳转到登录页
3. 点击"使用演示账号"按钮，自动填充：
   - 账号: `demo`
   - 密码: `demo123`
4. 点击"登录"按钮

### 浏览设备列表
1. 登录成功后进入设备列表页
2. 可以看到 4 台模拟设备
3. 点击筛选标签查看不同状态的设备
4. 点击任意设备卡片进入设备详情

### 设备详情
1. 查看设备的 3D 视图（占位符）
2. 切换到"监测总览"查看 KPI 数据
3. 切换到"曲线"查看图表（占位符）

### 主界面导航
1. 点击右下角的 FAB 按钮（或从设备列表点击进入）
2. 进入主界面，包含 4 个底部标签：

#### 标签 1: 3D设备
- 查看 3D 模型（占位符）
- 查看 KPI 指标卡片
- 点击提示查看部件详情抽屉
- 查看最近告警

#### 标签 2: 实时监测
- 查看连接状态
- 选择时间范围
- 查看大型温度 KPI
- 查看其他指标（电压、电流、功率、电能）
- 查看最新事件

#### 标签 3: 告警/工单
- **告警中心**:
  - 按级别和状态筛选
  - 查看告警列表
  - 点击告警卡片进入告警详情
  - 在详情页可以查看证据、建议等
- **工单列表**:
  - 按状态和时间筛选
  - 查看工单列表
  - 点击工单卡片进入工单详情
  - 在详情页可以勾选清单、查看时间线等

#### 标签 4: 我的/设置
- 查看个人信息
- 切换通知开关
- 查看阈值设置
- 点击"已绑定设备"返回设备列表
- 点击"退出"返回登录页

## 页面导航流程

```
启动页
  ↓
登录页
  ↓
设备列表 ←→ 主界面 (底部4个标签)
  ↓           ├─ 3D设备
设备详情      ├─ 实时监测
  ├─ 3D视图   ├─ 告警/工单
  ├─ 监测总览 │  ├─ 告警中心 → 告警详情
  └─ 曲线     │  └─ 工单列表 → 工单详情
              └─ 我的/设置
```

## 常见问题

### Q: 如何修改颜色主题？
A: 编辑 `lib/theme/app_theme.dart` 中的 `AppColors` 类

### Q: 如何添加新页面？
A: 
1. 在 `lib/screens/` 创建新页面文件
2. 在 `lib/routes/app_routes.dart` 添加路由
3. 使用 `Navigator.pushNamed()` 导航

### Q: 如何替换模拟数据？
A: 编辑 `lib/mock_data/mock_data.dart`

### Q: 如何集成真实 API？
A: 创建 API service 层，替换 mock_data 的调用

### Q: 3D 和图表如何实现？
A: 建议使用以下库：
- 3D: `flutter_cube` 或 `model_viewer`
- 图表: `fl_chart` 或 `syncfusion_flutter_charts`
